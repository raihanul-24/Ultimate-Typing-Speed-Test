# Ultimate Typing Speed Test

## Project Overview
A Java Swing application designed to test and improve typing speed and accuracy with customizable difficulty levels, real-time feedback, and leaderboard tracking.

## Features
- Real-time typing test with WPM and accuracy tracking
- Customizable timer and difficulty levels
- Leaderboard and result export
- Light/Dark themes and pause/resume functionality

## Tech Stack
- Java (Swing)
